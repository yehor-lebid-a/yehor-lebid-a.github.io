# Apester Test Pages

Static site for testing [Apester](https://apester.com) web SDK embed units in various page layouts.

## Structure

```
stg/                    Staging SDK test pages
  ├── center/           Widget centered on page
  ├── sidebar/          Widget in left sidebar
  └── bottom/           Widget below fold (scroll required)
archive/                Legacy test pages
```
